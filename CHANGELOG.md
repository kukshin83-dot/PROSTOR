# Changelog

Все значимые изменения в проекте «PROSTOR» в рамках учебной практики (ПМ.04).

## [1.1.0] - 2026-06-14

### Added
- **Новая функция**: сортировка товаров по убыванию цены через параметр `sort=price` (эндпоинт `GET /api/products`).
- **Валидация входных данных**: в `ProductRequest` добавлены аннотации `@NotBlank`, `@NotNull`, `@Positive`, `@Size`; в контроллере использована `@Valid`.
- **Мониторинг**: настроен Spring Boot Actuator (эндпоинты `health`, `info`, `metrics`).
- **Документация**:
  - Обновлён `README.md` (добавлены требования к системе, переменные окружения, раздел troubleshooting, описание параметра `sort`).
  - Создан `CHANGELOG.md`.

### Changed
- **Обновлены зависимости**: JJWT обновлён с 0.11.5 до 0.12.6 (в `build.gradle`).
- **Рефакторинг**: удалены неиспользуемые импорты, улучшена читаемость кода.

### Security
- Валидация DTO предотвращает сохранение некорректных данных (отрицательная цена, пустое название и т.д.).
- CSRF-защита (Spring Security) оставлена включённой по умолчанию.

### Performance
- Добавлены индексы на `seller_id` и `category_id` – ускоряют запросы фильтрации (миграция `V13__add_indexes.sql`).

## [1.0.0] - 2026-05-16
- Начальная версия проекта (исходный код маркетплейса PROSTOR).
